Box Model:

Note: when we set width and height we only set width and height of the content area, but when we need to calculate full size of element we should add padding, border and margins also.We can see the diffrences in below examples,"Paragraph one" has only width 400px and "Paragraph two" also has width 400px but including paddings,borders,and margins. So "Paragrph two" is box model example.

Here is width calculation for "Paragraph two":

360px (width)+ 10px (left + right padding)+ 10px (left + right border)+ 0px (left + right margin)= 400px
