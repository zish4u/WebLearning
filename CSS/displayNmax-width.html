Css Display

The "display" is most important css propert for controlling layout. The display property specifies if/how an element is displayed. Every HTML element has a default display value depending on what type of element it is. The default display value for most elements is block or inline.

A block-level element always starts on a new line and takes up the full width available (stretches out to the left and right as far as it can).see the example below-

"div, h1-h6, p, form, header, footer, section" are the block level elements.This para is in div which is also a block level elemnt thats why its taking full width.

An inline element does not start on a new line and only takes up as much width as necessary.See the example below.
"span, a, img" are the example of inline elements.This para is in span which is also a inline elemnt thats why its not taking full width.
