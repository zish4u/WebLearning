Box Model:

Note: when we set width and height we only set width and height of the content area, but when we need to calculate full size of element we should add padding, border and margins also.See the examples below, box model one has only width 600px and box model two also has width 600px but including paddings,borders,and margins.

Here is width calculation for box model two:

500px (width)+ 25px (left + right padding)+ 25px (left + right border)+ 0px (left + right margin)= 600px
